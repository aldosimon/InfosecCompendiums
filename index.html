Infosec Compendiums

The infosec compendium project aims to inventories topics of information security, including tools, scripts, cheat sheets, and knowledge bases, among other things. The project started because from time to time I need access to a specific infosec project but can't remember its name to search. So, the main goal of the project is to provide a one pager with an easy way to filter or search.
